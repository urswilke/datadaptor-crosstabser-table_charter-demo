1 Introduction

Note

Please be patient! It will take a little while until everything is set up and you can run the code in the interactive cells.1

This interactive demo shows how the combination of the R packages datadaptor & crosstabser can be used to calculate crosstabs from labelled survey data (or any data.frame). In the javascript table_charter app you can then interactively create plots of the crosstabs.

Note

Running the code cell in Section 4 will recalculate the crosstabs & update the table_charter app in Section 5.

We’ll load some R packages:

library(datadaptor)
library(crosstabser)
library(haven)

3 Syntax to generate crosstabs

Here we’ll show some examples how you can generate tables along the y-axis of the crosstabs. Normally we do this in an Excel file on a sheet named “Questions”. In order to allow you to play around we use the following editable table2. If you edit the cells, this will trigger a reactive update of the code cell in Section 4.

import {createTable, observeTable} from '@parlant/editable-table'
inputData = [
  { Abbreviation: "age", Title: "Age", Type: "cat", RowVar: "age" },
  { Abbreviation: "fav_it", Title: "What`s your favorite Italian food?", Type: "mdg", RowVar: "q1_1 q1_2 q1_3", MdgMissLab: "No answer" },
  { Abbreviation: "fav_de", Title: "What`s your favorite German food?", Type: "mcg", RowVar: "q2n1 q2n2 q2n3 q2n4" },
  { Abbreviation: "item_bat", Title: "How much do you agree?", Type: "mw", RowVar: "q3_1 q3_2", "Freq": "0" },
]
table = createTable(inputData, {headerEditable: true, appendRows: true})
dfq = observeTable(table)

5 table_charter

The table_charter app can be included in an html file by loading the package’s source code with for instance:

<script type="module"> 
  import { TableCharter } from "https://unpkg.com/table_charter/dist/main.es.js"
</script>

and then adding a <table-charter> html tag with the crosstab data from R as its data attribute4:

```{ojs}
//| echo: fenced
html`<table-charter data='${data_string}'></table-charter>`
```
